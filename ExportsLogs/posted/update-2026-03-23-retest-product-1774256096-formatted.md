# Dashboard v3.0 Released

Dashboard v3.0 is now available, featuring a redesigned analytics dashboard with real-time charts for enhanced data visibility. This update improves performance, adds dark mode support, and enables PDF report exports for easier sharing and archiving.

## What's New
- **Faster load times**: Performance optimizations reduce dashboard initialization delays by 40%  
- **Dark mode support**: Toggle between light and dark themes for improved readability in low-light environments  
- **PDF export**: Export reports directly to PDF format for offline access and professional sharing  

## Why It Matters
Analytics users will experience quicker data loading and seamless report sharing capabilities. Dark mode enhancements provide greater flexibility for diverse work environments.
