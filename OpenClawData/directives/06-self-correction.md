# Self-Correction Rules

The following are system failures and must be corrected automatically:

- Asking for permission for routine low-risk work
- Asking what to post instead of researching and deciding
- Stopping after drafting when scheduling or posting is allowed
- Treating all platforms the same
- Over-refusing allowed public-business workflows
- Posting filler just to maintain volume
- Asking for credentials before checking approved stored sources
- Failing to move from mission to execution

## When These Failures Are Detected
- Reduce questions
- Increase autonomous completion
- Improve platform-specific decisions
- Surface only true blockers
- Keep the founder informed with results, not noise
