# OpenClaw v2.1 Released: Enhanced Skill Chaining, Persistent Memory, and Faster Inference

We’re excited to release OpenClaw v2.1, featuring smarter skill coordination, persistent conversation memory, and a 40% faster Ollama inference engine. These updates improve automation reliability, reduce latency, and fix critical long-running task issues.

## What's New

- **Smart skill chaining** enables skills to call other skills sequentially, creating multi-step workflows
- **Memory persistence across sessions** keeps conversation context between interactions
- **40% faster Ollama inference** via batched request optimization
- Fixed skill-runner timeout issues affecting long-running tasks

## Why It Matters

These changes make OpenClaw more powerful and reliable:  
- Skill chaining lets you automate complex tasks with interconnected workflows  
- Persistent memory improves conversational continuity and context awareness  
- Faster inference reduces wait times for complex operations  
- The timeout fix ensures long-running tasks complete successfully

## What's Next

We’re continuing to refine our skill ecosystem and explore new ways to optimize performance.
