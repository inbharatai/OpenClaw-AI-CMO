# Claude Code CLI Tool Launched for Enhanced Developer Productivity

Anthropic has released Claude Code, a command-line interface (CLI) tool enabling developers to leverage Claude’s AI capabilities directly in their terminal environments. This tool is now available to all Claude Pro and Team subscribers, offering seamless integration for coding tasks without leaving the terminal.

## What's New

- **Claude Code CLI Tool**: A new CLI tool allowing developers to use Claude’s AI for coding tasks within terminal environments.  
- **Terminal-Native Workflow**: Supports file editing, shell commands, and Git operations directly through the CLI.  
- **Access for Pro/Team Subscribers**: Available to all Claude Pro and Team subscribers, expanding AI-assisted development capabilities.  

## Why It Matters

Claude Code bridges the gap between AI-powered development and terminal-native workflows, enabling developers to harness AI assistance without switching contexts. This integration streamlines productivity for developers who prefer command-line interfaces, making AI-assisted coding more accessible and efficient.

## What's Next

Anthropic plans to expand Claude Code’s capabilities in future updates, focusing on deeper integration with development workflows and additional language support.
