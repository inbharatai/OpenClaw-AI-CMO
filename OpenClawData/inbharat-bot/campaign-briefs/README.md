# Campaign Briefs Lane

This lane stores structured campaign briefs generated for handoff from InBharat Bot to OpenClaw.
Each brief follows a strict JSON format for machine-readable processing.

## Contents
- Campaign brief JSON files
- Brief status tracking (draft/approved/handed-off/executed)
- Supporting evidence and assets referenced by briefs
- Brief templates and examples

Every brief must pass approval before being handed off to OpenClaw for execution.
