# Discovery Lane

This lane wraps the scanner and gap-finder modules to orchestrate external discovery workflows.
It coordinates problem scanning, opportunity identification, and market intelligence gathering.

## Contents
- Discovery run outputs (JSON/Markdown reports)
- Aggregated scanner + gap-finder results
- External signal captures (news, trends, policy changes)
- Discovery queue files for pending analysis

This lane feeds into india-problems/, ai-gaps/, and funding/ lanes.
