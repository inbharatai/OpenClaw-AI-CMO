# Handoffs Lane

This lane manages structured handoffs from InBharat Bot to OpenClaw and other systems.
Handoffs include campaign briefs, content drafts, intelligence reports, and action items.

## Contents
- Handoff manifests (JSON) listing what is being handed off
- Status tracking (pending/accepted/in-progress/completed)
- Handoff logs and receipts
- Cross-system reference mappings

Each handoff is timestamped and tracked for accountability.
