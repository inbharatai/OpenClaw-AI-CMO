### 3. Phoring — Decision Intelligence Platform
**Repo:** github.com/inbharatai/phoring
**Status:** Development
**Stack:** Python 3.11+, Flask, Next.js, Zep (knowledge graph), OASIS
**License:** MIT
**Demo:** phoring.onrender.com

Converts unstructured documents into multi-agent simulations with source-cited forecasts.

5-step pipeline:
1. Graph Build — Extract domain ontology, construct Zep knowledge graph
2. Environment Setup — Configure LLM provider, validators, speed mode
3. Simulation — Run parallel OASIS simulations (Twitter, Reddit)
4. Report Generation — Source-cited forecasts with confidence scoring
5. Interactive Q&A — Follow-up using graph tools + web intelligence

Features:
- Knowledge graph from any document
- Web intelligence enrichment (Serper API, Event Registry)
- Multi-AI consensus validation (GPT-4o, Claude, Gemini)
- ReACT-style Report Agent with inline citations
- Security: input validation, path traversal prevention, XSS protection
