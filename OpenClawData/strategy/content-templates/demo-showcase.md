# Demo / Showcase Template

## Objective
Show {product} actually working. Proof over claims.

## Hook Styles
- "Watch {product} do {thing} in real time."
- "Here's {product} in action — no scripts, no demos, real usage."
- "Built this. It works. See for yourself."

## Structure
1. **What you're about to see**: Set expectations
2. **The demo**: Screenshot, video brief, or walkthrough
3. **What to notice**: Point out the key capability
4. **Context**: Why this matters, what's coming next
5. **CTA**: Try it, see more, follow for updates

## Image/Video Style
- Real product screenshots (current UI, not mockups)
- Screen recording brief (for video)
- Annotated screenshot with callouts
- Before/after split showing the capability

## Rules
- Must show REAL product, not a mockup pretending to be real
- If showing a prototype, say "prototype" or "in development"
- No fake data in screenshots — use realistic but clearly test data
