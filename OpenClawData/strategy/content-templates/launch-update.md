# Launch / Update Template

## Objective
Announce something new — a release, feature, milestone, or significant update. Build excitement without overhyping.

## Hook Styles
- "Just shipped: {feature/update}. Here's what's new."
- "After {timeframe} of building, {product} now does {capability}."
- "{Product} v{version} is live. The biggest change: {one thing}."

## Structure
1. **Announcement**: What shipped, in one clear sentence
2. **What's new**: 2-3 specific changes or capabilities
3. **Why it matters**: Impact for the user
4. **What's next**: Brief forward-looking note
5. **CTA**: Try it, see it, star it

## Tone
- Confident but measured
- Celebrate without overhyping
- Specific: "added Hindi support" not "major update"
- Honest about scope: "v0.2 alpha" if that's what it is

## Image Style
- Product screenshot showing the new feature
- Before/after comparison
- Clean announcement card with version number

## Platform Notes
- **Discord**: Post first here (community gets it first)
- **X**: Quick announcement tweet, link to details
- **LinkedIn**: Longer narrative about the journey + what's new
- **Instagram**: Visual-first — screenshot or announcement card
