# How It Works Template

## Objective
Educate the audience on how {product/feature} works. Position as helpful, not promotional.

## Hook Styles
- "How does {product} actually work? Here's a simple breakdown."
- "3 steps to {outcome} using {product}."
- "Ever wondered how AI can {capability}? Here's the architecture."

## Structure
1. **Hook**: Question or curiosity driver
2. **Step 1**: First thing that happens (simple language)
3. **Step 2**: The AI/core logic part (demystified)
4. **Step 3**: The output/result the user gets
5. **Why it matters**: Connect to real user benefit
6. **CTA**: Try it yourself

## Image Style
- Flow diagram (Step 1 → Step 2 → Step 3)
- Architecture diagram (simplified)
- Carousel: one step per slide
- Screenshot walkthrough

## Best Formats
- **Instagram**: Carousel (one step per slide)
- **LinkedIn**: Text post with diagram image
- **X**: Thread (one step per tweet)
- **Website**: Full article with embedded visuals
