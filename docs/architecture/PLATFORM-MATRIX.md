## Status Legend

| Status | Meaning |
|--------|---------|
| **Auto-post** | SocialFlow posts directly via browser automation |
| **Auto (queue)** | Content queued as files, published via your website pipeline |
| **Draft only** | Content generated as drafts, you copy-paste to post |
| **Brief only** | Briefs generated, you use them to create in-platform |
| **Manual** | High-risk platform, always post manually |
| **Later** | Not yet integrated, planned for future versions |

## Platform-Specific Guardrails

### LinkedIn
- No aggressive sales language
- Professional tone required
- Max 3000 characters per post
- Include relevant hashtags (3-5)

### X / Twitter
- Max 280 characters (or thread format)
- No mass following/unfollowing
- Respect rate limits strictly

### Facebook
- Avoid clickbait language
- No misleading thumbnails
- Comply with community standards

### Instagram
- Requires image/visual content
- Max 2200 characters for captions
- 30 hashtag limit

### Discord
- Use proper formatting (markdown)
- Don't spam multiple channels
- Respect server-specific rules

### Reddit
- **NEVER spam**. Reddit bans aggressively
- Follow subreddit rules strictly
- Value-first content only
- Manual posting strongly recommended

### Medium / Substack
- Long-form quality required
- No thin/low-effort content
- Proper formatting and structure
