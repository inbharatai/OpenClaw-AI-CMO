# CodeIn Launches AI-First Coding Platform with Zero Telemetry for Indian Developers

CodeIn now offers AI-powered coding assistance and education designed specifically for Indian developers and students, with zero telemetry and no data collection. Your code remains entirely local, running on Ollama models without sending any data to external servers.

## What's New

- **Local AI Coding Platform**: CodeIn runs entirely on your machine using Ollama models, ensuring no code leaves your device.
- **Zero Telemetry Policy**: No data collection or tracking—your code, your data, your control.
- **Privacy-Focused for Indian Users**: Built with Indian developers and CS students in mind, prioritizing privacy and affordability.
- **Affordable Alternative**: A cost-effective solution for those who can't afford $300/year for GitHub Copilot.
- **Website Launch**: [codein.pro](https://codein.pro) now hosts the platform for seamless access.

## Why It Matters

Indian developers and students who prioritize privacy and need an affordable coding assistant now have a trusted alternative to cloud-based tools. By keeping code local and data private, CodeIn empowers users with control and security without compromising on AI-driven productivity.

## What's Next

We’re expanding our focus on local AI infrastructure and privacy-first features for Indian developers in upcoming updates.
