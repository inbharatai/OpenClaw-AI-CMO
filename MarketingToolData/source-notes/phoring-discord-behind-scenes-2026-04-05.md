# Behind the Scenes: Building the Multi-Agent Knowledge Graph

Product: Phoring
Platform: discord
Type: behind-the-build
Priority: high

Phoring is building a decision intelligence platform that converts documents into multi-agent simulations with source-cited forecasts.

The core innovation is the multi-agent knowledge graph — a system where multiple AI agents collaborate to analyze documents, build relationship graphs, run simulations, and generate forecasts with citations back to the original sources.

The 5-step pipeline:
1. Graph Build — Extract entities and relationships from documents
2. Environment Setup — Configure simulation parameters
3. Simulation — Run multi-agent scenario analysis
4. Report Generation — Produce source-cited insights
5. Interactive Q&A — Ask follow-up questions with evidence trails

Tech stack: Python, Flask, Next.js, Zep, OASIS
