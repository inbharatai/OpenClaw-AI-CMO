# Content Templates

Templates for generating platform-specific content.

## Available Templates

Place template files here for:
- `linkedin-post.md` — LinkedIn post template
- `x-post.md` — X/Twitter post template
- `discord-announcement.md` — Discord announcement template
- `newsletter.md` — Newsletter template
- `website-update.md` — Website update template
- `weekly-roundup.md` — Weekly roundup template
- `video-brief.md` — HeyGen video brief template
- `image-brief.md` — Image generation brief template

Templates use `{{variable}}` placeholders that are filled by skills during content production.
