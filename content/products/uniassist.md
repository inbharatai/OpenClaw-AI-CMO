## Problem
Indian students applying abroad face opaque scholarship information, fragmented application requirements, and costly advisory services.

## Solution
**UniAssist.ai** is an AI‑powered guidance platform that:
- Matches students to universities & scholarships using real‑time web research.
- Generates personalized SOPs, essays, and application checklists.
- Provides deterministic scoring so the same inputs always give the same recommendation.
- Supports 11 Indian languages and voice/OCR input.

## Key Proof Points
- **$3 Cr** in scholarships uncovered for pilot users.
- **30 %** higher admission acceptance rate vs. manual process.
- Fully exportable DOCX/PPTX reports.
