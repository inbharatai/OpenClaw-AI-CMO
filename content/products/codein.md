## Problem
Developers in India need a coding assistant that works offline, respects language diversity, and incurs no recurring fees.

## Solution
**CodeIn.pro** is an open‑source AI coding IDE that:
- Supports **22 Indian languages** with voice input (via Whisper/Vosk).
- Runs **fully offline** using llama.cpp (zero telemetry).
- Offers a **Cursor‑style UI** with multi‑model support (60+ LLM providers).
- Exports code snapshots to Git without leaving the machine.

## Proof Points
- **$0** monthly cost – saves $120–$480 vs. GitHub Copilot/Claude.
- Used in **3 + state government coding bootcamps**.
- **100 %** of data stays on‑device (privacy‑first).
