# InBharat Core (placeholder)

This folder will eventually host the shared libraries that all InBharat products will consume:

- **inbharat‑nlp** – multilingual NLU, Whisper/Vosk wrappers.
- **inbharat‑search** – unified Serper / Google‑search API.
- **inbharat‑ui** – React component library (brand header, language switcher, colour theme).
- **inbharat‑auth** – Supabase configuration and JWT utilities.
- **inbharat‑metrics** – OpenTelemetry exporter.

For now it is empty; the actual code will be added when the architecture is finalised.
