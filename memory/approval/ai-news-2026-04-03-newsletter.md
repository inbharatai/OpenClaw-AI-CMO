## 🗞️ Today’s Highlights

### 1. Anthropic – Claude Code Source Leak
Anthropic announced that an npm packaging error exposed the internal source code of its Claude Code assistant (≈2,000 TS files, 512k lines). No customer data was compromised. The offending package version was pulled and a security audit is underway.

### 2. Meta – AI‑Native Transformation & Layoffs
Meta disclosed a 700‑person layoff and unveiled internal AI‑adoption targets: by late‑2025, 55% of code changes must be AI‑assisted. Engineers are expected to use tools like **DevMate**, **Metamate**, and **Google Gemini**.

### 3. OpenAI – Record‑Breaking Funding
OpenAI closed a $122 B financing round, pushing its valuation to **$852 B**. The capital will fuel further scaling of ChatGPT, Whisper, and upcoming multimodal models.

### 4. DeepMind – Gemma 4 Release
DeepMind introduced **Gemma 4**, its most capable open‑source model yet, engineered for advanced reasoning and agentic workflows. It promises higher intelligence per parameter, broadening access to high‑end AI capabilities.

### 5. Microsoft – Azure AI Copilot Expansion
Microsoft expanded Azure AI Copilot across its cloud services, adding real‑time code generation, data analysis, and Power Platform integration. Early adopters report a ~30% boost in developer productivity.

### 6. NVIDIA – H100 X GPU
NVIDIA launched the **H100 X** accelerator, delivering up to **2×** the AI performance of the previous H100. The new hardware is paired with an upgraded software stack optimized for transformer training.

### 7. Hugging Face – Hub 2.0
Hugging Face rolled out **Hub 2.0**, featuring versioned model repositories, community‑curated datasets, and integrated inference endpoints to streamline AI deployment.
